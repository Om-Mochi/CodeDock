
<h1>CodeDock</h1>

![CodeDock](CodeDock/icons/github_demo/win_demo.png)

CodeDock is a custom code editor and docking framework built with Python and PyQt6. It includes a VS Code-inspired editor experience, a custom docking system, Language Server Protocol (LSP) integration, a minimap, diagnostics, autocomplete, and workspace management. The project was created to explore desktop application architecture and advanced editor development.

«⚠️ This project is primarily a learning project and is not intended to represent production-quality architecture or code organization.»

---

Advanced Docking System

- Custom docking architecture

<h2>What I Learned</h2>

CodeDock became one of my largest personal projects and taught me many important lessons about software development:

- Desktop application architecture
- Custom widget development
- Event-driven systems
- Multi-threaded programming
- Language Server Protocol integration
- UI framework design
- Project organization and maintainability

As the project grew, the architecture became increasingly difficult to maintain. Looking back, there are many design decisions I would approach differently today. The codebase contains experimental ideas, inconsistent patterns, and areas that would benefit from significant refactoring.

However, those challenges became valuable learning experiences and helped me better understand scalable software design.
